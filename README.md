# Xander Budge MultiTool

## Getting Started
### Install

#### MultiTool
* MultiTool is Built for windows
* Clone this repo
* Ensure you have python 3.11 or greater for multitool itself
  * 3.12 or greater for calcure calendar 
#### AutoHotKey V2
* Install AutoHotKey V2: 
  * https://www.autohotkey.com/v2/
* Open folder where MultiTool is installed
* In `MultiTool.ahk` Replace ```"C:\Users\dylan\PycharmProjects\Multitool\MultiTool.py" ``` with your path to MultiTool.py
### Calcure Calander package
* Install calcure calendar package (requres python 3.12)
  * https://anufrievroman.gitbook.io/calcure
  * https://github.com/anufrievroman/calcure
### Ensure AHK shortcurt runs on startup
* Open folder where MultiTool is installed
* Create Shortcut for `MultiTool.ahk`
  * Select `MultiTool.ahk`
  * Right click on `MultiTool.ahk`
  * Show more options
  * Create Shortcut
* Cut the shortcut by with ctrl + x
* Press Win + r
* Type shell:startup
* Press Enter
* Paste `MultiTool.ahk` shortcut  into that startup folder with ctrl + v


### How To Use
* Either restart or double-click on `MultiTool.ahk` for first run
* Press ctrl + shift + m to open multitool
  * * NOTE: You may have to wait for other apps to run on startup first, before the shortcut will be active
* navigate menus with numbers
* Refer to calcure documentation for help with calendar
